# Outbox

`vault/outbox/` stores generated deliverables and export packages.

Examples:

- context export packages
- NotebookLM source packs
- workshop packs
- stakeholder reports
- handoff/continuity packages
- generated documentation bundles

Any output that should be remembered later should also be referenced from `vault/wiki/index.md` or another curated wiki note.
